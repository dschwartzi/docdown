Features

Browse Your Drive

Navigate My Drive, Shared with me, Starred, and Recent files in a clean sidebar.

Search Everything

Full-text search across all your files and shared drives. Paste a URL to grab any doc directly.

Clean Markdown

Google Docs, Sheets, and Slides are converted to clean Markdown — no CSS noise or formatting junk.

Bulk Download

Select All and download an entire folder of docs as .md files in one click.

URL Paste

Paste any Google Doc or Drive folder URL into the search bar to instantly access it.

Dark & Minimal

Modern dark interface that stays out of your way. Built with Electron for all platforms.

Getting Started

1

Download & Install

Grab the installer for your platform above and run it.

2

Choose a Sync Folder

Pick a local folder where your Markdown files will be saved.

3

Sign In with Google

Your browser will open to authorize read-only access to your Google Drive.

4

Browse & Download

Navigate your Drive, select files, and pull them down as clean Markdown.
